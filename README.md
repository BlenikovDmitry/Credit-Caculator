
Реализация кредитного калькулятора с помощью библиотеки streamlit
Что умеет:

- cчитывание суммы кредита, процентной ставки, срока кредита и типа платежа по кредиту (аннуитетный и дифференциальный);
- вычисление ежемесячного платежа по кредиту (для аннуитетного типа);
- формирование графика платежей по кредиту в виде таблицы со следующими столбцами:
  остаток долга,
  ежемесячный платеж весь,
  процентная часть,
  долговая часть,
  остаток долга на конец периода;

- считывание даты первого платежа и добавление в таблицу даты каждого из платежей;
- обработку случаев неправильного ввода;
- использование механизмов условного рендеринга (if/else, st.expander(), st.rerun() и st.stop())

- два варианта интерфейса
- файл config.py для подгрузки параметров в программу
- файл config.toml для примитивной кастомизации интерфейса
- вычисление переплаты
- градация ставок в зависимости от категории клиента
